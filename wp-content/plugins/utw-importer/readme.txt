== Description ==

Import Ultimate Tag Warrior tags into WordPress tags.

== Installation ==

1. Upload the `utw-importer` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to the Tools -> Import screen, Click on Ultimate Tag Warrior
